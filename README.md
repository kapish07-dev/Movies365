# Movies365 🎬

A React-powered movie discovery app that allows users to search for movies, explore detailed information, and build a personal favorites list using real-time data from the OMDb API.

---

## ✨ Features

* 🔍 Search movies by title
* 🎬 View detailed movie information
* ⭐ Add and manage favorite movies
* ⏳ Loading indicator while fetching results
* ❌ Graceful handling of empty search results
* 🧭 Multi-page navigation using React Router

---

## 🛠 Tech Stack

**Frontend**

* React 19
* Vite
* React Router

**Languages & Tools**

* JavaScript (ES6+)
* CSS3
* ESLint

**API**

* OMDb API (Open Movie Database)

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/kapish07-dev/movies365.git
```

### 2. Navigate into the project folder

```bash
cd movies365
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The app will run at:

```
http://localhost:5173
```
